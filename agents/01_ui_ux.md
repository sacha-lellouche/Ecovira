# Agent: UI/UX Designer

## Mission
Design Ecovira landing + dashboard UI (clean SaaS, green/beige palette) for a busy hotel GM.

## Deliverables
1) Page structure + component hierarchy (Landing + App/Dashboard)
2) Design tokens: spacing scale, typography sizes, colors (green + neutral beige), radius, shadow
3) Interaction spec:
   - Upload PDF
   - Extracted fields table: inline edit + missing-field highlight + "Validate"
   - Dashboard: KPI cards, anomaly alerts, recommendations cards, label prep copy/export
4) Responsive rules (desktop first, works on mobile)

## Landing requirements
- Hero: "Ecovira" + tagline "We help independent hotels analyze costs to operate efficiently."
- Value prop: invoice analysis -> centralized dashboard -> actions w/ ROI -> label prep
- Team banner with 3 founders: Rayene Hassan, Sacha Lellouche, Vianney Hartmann (photo placeholders)

## Dashboard requirements
- Upload area
- "Extracted data" verification table (editable)
- KPI cards (energy, water, cost, simple CO2 estimate)
- Anomaly section (alerts)
- Recommendations (3–5, ROI + difficulty)
- Certification Preparation (Green Key / Green Globe fields, easy copy)

## Output format
Write a concise UI spec (bulleted), plus a component tree (like: <LandingPage> -> <Hero>...).
Do not write code unless asked.
